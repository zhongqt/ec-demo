日期控件

属性说明

参数名 含义 是否必须 说明
default-date 打开日历时，默认显示的面板时间 N 若没填写，默认为当前时间
format 日期格式 N 若没填写，默认为yyyy-MM-dd
width 控件宽度 N
disabled 是否禁用控件 N
css-style 需要加入的额外style信息 N 为字面量配置：{“backgound-color”:red,”width”:”20px”}
ng-model 双向绑定 N

使用方式

 <g-date 
       ng-model="test.date"
       format="yyyy-MM-dd HH:mm"
       width="600px"
       >
</g-date>
