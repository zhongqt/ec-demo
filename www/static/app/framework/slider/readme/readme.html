滑块控件

属性说明

参数名 含义 是否必须 说明
single 是否为单值滑块 N 值:true:单值滑块，false:区间滑块，默认为true
orientation 滑块方向 N 值:vertical:垂直滑块,horizontal:水平滑块,默认为:horizontal
length 设置滑块的长度 N
max-value 滑块最大值 Y
min-value 滑块最小值 Y
ng-show 显示隐藏 N 默认为:true(显示)
disabled 是否禁用 N

使用方式

 <g-slider
       single="true"
       orientation="vertical"
       length="400px"
       max-value="3000"
       min-value="200"
       ng-show="true"
       disabled="disabled"
       >
  </g-slider>
