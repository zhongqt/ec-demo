下拉按钮

属性说明

参数名 含义 是否必须 说明
disabled 禁用按钮 N 只要有设置则禁用。如disabled="true"或者disabled="disabled"
button-icon 按钮图标 N 按钮图标class样式
width 设置控件宽度 N 只能在主指令中设置.单位有%:百分比;in：英寸;cm：厘米;mm：毫米;pt:磅数;pc:12点活字;ex:字母高度的一半;em:字体高度;px：像素(默认)
is-button 是否为按钮分裂下拉按钮 N 是否为button分裂下拉按钮;true(默认：按钮);false:a标签按钮
display 菜单按钮上显示文字 Y
click-event 配置点击事件 N 当href-prop和click-event同时配置时，只有click-event函数返回true时，才会跳转到href-prop的页面。
href-prop 配置的超链接地址 N
ng-show 配置控件的显示或隐藏 N
role 是否为分隔线 N role="separator"为分隔线

使用方式

 <g-dropdown-btn
       g-width="130px"
       display="确定提交"
       click-event="clickMajorBut('确定按钮')"
       href-prop="http://www.baidu.com"
       ng-show="true">
       <g-dropdown-item
           disabled="disabled"
           g-width="130px"
           display="确定提交"
           click-event="clickMajorBut('确定按钮')"
           href-prop="http://www.baidu.com"
           ng-show="true">
       </g-dropdown-item>
  </g-dropdown-btn>
