星级控件

属性说明

参数名 含义 是否必须 说明
disabled 是否禁用控件 N 控制元素是否禁用。disabled:只要设有此属性则为禁用。不设此属性为启用。
readonly 只读 N 设置为true时，为只读控件，不可点击。可以通过ng-model来改变
ng-model 双向绑定 N
ng-show 显示隐藏 N 值:true:显示，false:隐藏
class-name 额外样式 N 添加到控件上的额外样式

使用方式

 <g-star
       ng-model="star.value"
       class-name="class"
       disabled="disabled"
       readonly="true"
       ng-show="true"
       >
  </g-star>
