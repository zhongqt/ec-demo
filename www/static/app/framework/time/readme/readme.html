时间控件

参数名 含义 是否必须 说明
default-time 默认时间 N 若没填写，默认为当前时间
pattern 时间格式 N 若没填写，默认为24小时制，即HH:mm,可配成hh:mm（12小时制）或HH:mm（24小时制）
width 控件长度 N
cssClass 需要加入的额外CSS-Class N
cssStyle 需要加入的额外style信息 N 为字面量配置：{“backgound-color”:red,”width”:”20px”}
disabled 是否禁用当前控件 N

使用实例：

<g-time 
pattern="hh:mm" 
width="300px" 
css-class="test-class" 
css-style="{'background-color':'#eee'}"  
ng-model="test.time"
default-time="'11:00'"
></g-time>
