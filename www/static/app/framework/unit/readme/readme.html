单位文本框

属性说明

参数名 含义 是否必须 说明
disabled 是否禁用控件 N
g-unit 控件名称 Y 如果只有后面的单位，可以不使用unit-after。把单位名称直接设在g-unit上(如:g-unit="$")
unit-before 单位放置前面 N 前面的单位名称(如:unit-before="kg")
unit-after 单位放置后面 N 后面的单位名称(如:g-unit="kg"),如果g-unit设了单位，再设unit-after的话。unit-after的单位会覆盖g-unit的单位。
g-dbc 全角转半角处理 N
g-numeric 数字 N 属性allow-float:是否允许输入小数,allow-negative:是否允许输入负号,如需保留小数位，则和g-precision一起使用
g-precision 保留小数位数 N 属性：rounding舍入模式:false:四舍五入;true：截断;不设置(默认):四舍五入
width 设置文本框宽度 N 值:单位有%:百分比;in：英寸;cm：厘米;mm：毫米;pt:磅数;pc:12点活字;ex:字母高度的一半;em:字体高度;px：像素(默认)
ng-model 双向绑定 N
allow-show 显示隐藏 N 值:true:显示，false:隐藏
placeholder 输入提示 N

使用方式

 <input
       ng-model="text.value"
       width="500"
       g-dbc
       g-unit="kg"
       unit-before="$"
       unit-after="$"
       g-numeric
       allow-float="true"
       allow-negative="true"
       g-precision="2"
       rounding="false"
       allow-show="true"
       placeholder="输入提示"
       >
  </input>
