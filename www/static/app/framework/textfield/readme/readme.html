属性说明

参数名 含义 是否必须 说明
disabled 是否禁用控件 N
g-case 大小写转换处理 N 值:upper:小写转大写,lower:大写转小写
g-dbc 全角转半角处理 N
g-height 设置文本框高度 N 值:单位有%:百分比;in：英寸;cm：厘米;mm：毫米;pt:磅数;pc:12点活字;ex:字母高度的一半;em:字体高度;px：像素(默认)
g-numeric 数字 N 属性allow-float:是否允许输入小数,allow-negative:是否允许输入负号,如需保留小数位，则和g-precision一起使用
g-precision 保留小数位数 N 属性：rounding舍入模式:false:四舍五入;true：截断;不设置(默认):四舍五入
g-trim 去空格 N 值：ltrim:去除左边空格;rtrim:去除右边空格;trim(默认值):去除左右空格。
g-width 设置文本框宽度 N 值:单位有%:百分比;in：英寸;cm：厘米;mm：毫米;pt:磅数;pc:12点活字;ex:字母高度的一半;em:字体高度;px：像素(默认)
ng-model 双向绑定 N
ng-show 显示隐藏 N 值:true:显示，false:隐藏
placeholder 输入提示 N

使用方式

 <input
       ng-model="text.value"
       g-width="600px"
       g-height="30px"
       g-dbc
       g-numeric
       allow-float="true"
       allow-negative="true"
       g-precision="2"
       rounding="false"
       g-trim="trim"
       ng-show="true"
       placeholder="输入提示"
       >
  </input>
